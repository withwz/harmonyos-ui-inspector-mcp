# 项目概述：鸿蒙 UI 查看 MCP 工具

## 目标

开发一个 MCP (Model Context Protocol) 服务器，让 Claude 能够查看鸿蒙系统的 UI 状态，包括：
- 窗口列表和信息
- UI 组件树结构
- 应用截图
- 能力（Ability）状态

## 使用场景

- **调试 RN 应用**：查看 React Native 在鸿蒙上的渲染状态
- **UI 分析**：分析页面结构和组件层级
- **性能排查**：结合窗口信息定位卡顿问题
- **自动化测试**：获取 UI 状态进行验证

## 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| `list_windows` | 列出所有窗口及其信息 | P0 |
| `get_ui_tree` | 获取 UI 组件树 | P0 |
| `get_abilities` | 获取应用（Ability）列表 | P1 |
| `screenshot` | 获取设备截图 | P1 |
| `find_element` | 根据属性查找 UI 元素 | P2 |

## 部署方式

- **位置**：`~/.claude/plugins/harmonyos-rn-ui/`
- **类型**：本地插件 + MCP 服务器 (stdio)
- **配置**：`.mcp.json` 文件
